# spotify-api-etl
This is a project that involves different technologies that extract data from a spotify API, transform it and load it.

The Technologies involved in this project includes:
- Python (AWS Lambda function)
- AWS Glue (Spark)
- S3 (Storage)
- SQL
- Snowflake (Snowpipe)
- Airflow (Orchestration)
